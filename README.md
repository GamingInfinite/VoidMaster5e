# Void Master 5e
Void Master is a webapp that is basically meant to supplant other tools like 5e.tools or D&D Beyond.  The reasons behind the development of this tool is for a couple of reasons.

1. D&D Beyond is a great website, and there character creator is phenominal, I just feel like their homebrew support could be a bit more robust.
2. 5e.tools, even though it's a great free resource for DM's and Players alike, it doesn't have a character creator, and basically bars any new player to using it.
3. There *is* a Android and Ios app called 5e Companion, and it has a lot of the resources that 5e.tools has as well as pretty good homebrew support, but doesn't have a good character creator in my opinion, as the user experience feels clunky and untested.

For all of these reasons I'm creating my own tool, that hopefully will have the best of all worlds: A great resource for D&D information at large, Robust Homebrew support, and a character creator that functions well, properly implements all features of any given class.
